AREAL WEB SERVER

Server commands

  • /config - show register and coil numbers
  • /status - show register and coil status
  • /read?coil=1 - read coil (1)
  • /read?hreg=1 - read holding register (1)
  • /read?ireg=1 - read input register (1)
  • /read?input=1 - read discret input (1)
  • /set?coil=1&value=0 - set coil (1) to value (0)
  • /set?hreg=1&value=23456 - set holding register (1) to value (23456)
  • /yyyymmddhh.csv - get archived register value for specified hour
  • /hungup - call hungup script
  • /alarm - call alarm script
